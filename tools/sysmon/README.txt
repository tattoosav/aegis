Sysmon Tool Directory
=====================

This directory is a placeholder for the Sysmon binary and its
configuration file.  At build time, PyInstaller will bundle the
contents of this directory into the distribution.

Required files (not shipped in source control):

  Sysmon64.exe     Microsoft Sysinternals Sysmon binary
  sysmonconfig.xml Custom Sysmon configuration for Aegis

Download Sysmon from:
  https://learn.microsoft.com/en-us/sysinternals/downloads/sysmon
